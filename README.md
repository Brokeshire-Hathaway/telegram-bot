# chatgpt-companion

## Requirements
- [Docker](https://docs.docker.com/engine/install)
- [Node](https://nodejs.org/en/download)
- [yarn](https://classic.yarnpkg.com/lang/en/docs/install)

## Getting Started
```sh
docker compose up
```
If you want to run in the background 
```sh
docker compose up -d
```
